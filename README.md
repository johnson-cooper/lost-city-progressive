## **REVIEW OF PLANS**

# **OVERVIEW**

Lost City Singleplayer Progressive is a mod built on the open-source Lost City 2004 RuneScape reimplementation. It introduces autonomous bots that progress alongside the player, leveling up and competing on the hiscores. These bots also simulate a living game world by engaging in activities such as PvP combat, trading, player interaction, rewarding, and even deceptive behaviors like scamming. The project is designed to be released as a standalone, easy-to-use package and will remain fully open source to ensure accessibility and community contribution.

# **GOALS**

1. Perseverance of nostalgic atmosphere that Runescape brought back then.  
2. Deliver a single player experience that is engaging, along with tools to broaden the experience.
